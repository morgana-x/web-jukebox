# Signed Jukeboxes (b1.7.3)
Clientside only babric mod that allows playing custom music on Jukeboxes with signs

Other clients with the mod should also be able to hear the custom music!

## Supported formats
Natively supports `wav, ogg, mid`

If FFMPEG is installed on your system, you can also use `mp3, flag, m4a, mkv, webm, opus` which will be converted to ogg

`.mid` midi files are technically supported but not recommended (playback cannot be cancelled until it finishes the song)

## How to use
Attach a sign (or multiple) to a Jukebox that has the music link you want to play

Then insert and eject any disc as usual

The "Paste Audio Link" button will paste any audio link onto the sign in your clipboard, 
it will also detect if part of it is contained in signs to the left, and paste the next remaining 60 or less characters.
